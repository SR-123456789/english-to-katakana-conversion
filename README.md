## 機能説明

英単語を引数にして、読み方をカタカナで返します。

```
const convert=require( "english-to-katakana-conversion")

console.log(convert("baseball"))
```
=>ベースボール

## インストール方法

```
npm install english-to-katakana-conversion
```

## 使い方

```
const convert=require( "english-to-katakana-conversion")

console.log(convert("baseball"))
```
